사용자 정의 데이터 타입

사용자 데이터 타입
사용자가 정의한 새로운 데이터 타입으로 자료구조 정의할 때 사용, 코드를 간편하게 해줌

struct 묶음 데이터 구조화된 데이터
enum 열거 데이터 여러 개의 항목을 집합으로
union 공용 데이터 타입
typedef 기존의 데이터 타입의 단순화한 별칭 정의

구조체의 정의
서로 다른 데이터형의 변수들을 하나로 묶어서 사용하는 기능
사용자 정의 데이터형을 만드는 방법 제공

구조체 사용 예
struct contact
{
char name[20];
char phone[20];
int ringtone;
}

함수 안에 정의하는 경우
정의된 함수에서만 사용할 수 있다.   

함수 밖에 정의하는 경우는 
소스파일 전체에서 사용할 수 있다.

----------------------------------------------사용자 정의형으로서의 구조체(1/2)
데이터형만 정의할 뿐 구조체 변수가 생성되는 것은 아니다

------------------------------------------------------------------------------------------
구조체 변수의 초기화
초기값이 멤버의 개수보다 부족하면 나머지 멤버들은 0으로 초기화됨

---------------------------------------------------------------------------------------
멤버의 개수보다 초기값을 더 많이 주면 컴파일 에러 난다.

초기값으로 {0} 을 지정하면 모든 멤버가 0으로 초기화된다.
